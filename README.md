# VagrantPi's Blog

紀錄一些生活札記

Blog 使用 [hyde](https://github.com/poole/hyde) 建置

Website: https://vagrantpi.github.io/

## install dependency

```
gem install bundler
bundle install
```

## build

```
$ jekyll build
```

## serve in local

```
$ jekyll serve
Auto-regeneration: enabled for '/VagrantPi.github.io'
   Server address: http://127.0.0.1:4000/
 Server running... press ctrl-c to stop.
```
